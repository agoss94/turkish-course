## Zahlen

Zahlen werden im türkischen nach einem regelmäßigen Muster gebildet.

| Zahlenwörter | Übersetzung |
| - | - |
| bir | eins |
| iki | zwei | 
| üç | drei |
| dört | vier |
| beç | fünf |
| altı | sechs |
| yedi | sieben |
| sekiz | acht |
| dokuz | neun |
| on | zehn |
| yirmi | zwanzig |
| otuz | dreizig |
| kırk | vierzig |
| elli | fünfzig |
| altmış | sechzig |
| yedmiş | siebzig |
| seksen | achtzig |
| doksan | neunzig |
| yüz | hundert |
| bin | tausend |

Nun werden zum bilden einer Zahl die Zahlwörter aneinander gereiht änhlich wie im deutschen. 

	1256 = bin iki yüz elli altı
